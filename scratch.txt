Demo Spring Cloud Gateway with Load Balancer:
    a. Start Gateway
    b. Start all three Sentence Applications
    c. Run following request with Postman:
        curl -X GET \
          http://localhost:8080/sentence-client/port \
          -H 'Content-Type: application/json' \
          -H 'Host: localhost:8080'
    d. Start and stop Sentence instances and verify correct load balancing

    b.  Run curl:
        curl -X GET \
          http://localhost:8080/sentence-client/sentence \
